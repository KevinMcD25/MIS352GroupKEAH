This is the READ ME For AdventureWV API

API 1 PlanAdd

This API adds a plan to the Travel Plan table
Inputs : HID, AID
OUTPUT PID, HID, AID

API 2 PlanDelete

This API deletes a plan in the Travel Plan table
Inputs : PID
OUTPUT : Drop(travel plan)



API 1 AddActivity2 (Andrew)
This API adds an activity in the Activities table.
Inputs: Aname, LID
Output: Aname, LID

API 2 SearchActivity (Andrew)
This API searches the Activities table for an activity
Inputs: Aname
Output: Aname
